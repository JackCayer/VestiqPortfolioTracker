📋 Import from Google Sheets
  1. Make sure your sheet columns are in order: Ticker | Shares | Avg Cost (no $ signs in the number cells)
  2. Easiest method: Click the green Share button → set to Anyone with the link → copy the URL and paste it below
  3. Alt method: File → Share → Publish to web → select sheet → choose CSV → Publish → copy URL
  4. Both URL formats are supported — the app converts them automatically
