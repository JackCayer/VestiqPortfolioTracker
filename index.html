☁️ Firebase Sync Setup

Syncs your data across all devices in real time — free forever for personal use.
Setup (5 min):
1. Go to console.firebase.google.com → Create project (disable Analytics)
2. Click Firestore Database → Create database → Start in test mode
3. Go to Project Settings ⚙️ → scroll to "Your apps" → click </> (Web)
4. Register app → copy the firebaseConfig object → paste below
5. Also go to Firestore → Rules and make sure rules allow read/write
